Detailed Description

Defines the T81Limb class, a 48-trit packed ternary integer.

This file provides T81Limb, a high-performance 48-trit (16-tryte) numeric type. It uses a Kogge-Stone carry-lookahead adder with function composition on carry maps to achieve high-throughput addition.
