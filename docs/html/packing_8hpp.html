Detailed Description

Balanced ternary packing helpers (8‑trit / 19‑trit encoders).

These functions encode balanced-trit sequences into tightly packed binary representations so we can compare the real storage cost versus naive Cell arrays. The current implementation supports the canonical 8-trit→13-bit and 19-trit→32-bit mappings described in the spec.
